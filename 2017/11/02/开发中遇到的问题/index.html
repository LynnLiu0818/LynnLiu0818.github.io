开发中遇到的问题

1. css

1. img

  1. 当给img标签加上宽和高，src链接为空时，img便签会存在边框，无法用border: 0;去掉
    解决办法：只给img标签加高度，设置图片cover，这样图片的大小就会等比缩放

2. 背景

  1. 弹出框想让背景无法滑动，需要给底层设置了“overflow:scroll”的标签设置为“overflow: hidden”

2. js

  1. 解决js算法算数来数字精度丢失的问题：
    Math.round(Math.floor(Math.round((xxx + xxx) * 10000) / 100) / 100);

2、 问题：H5页面在IOS端，弹性滑动会造成在滑动过程中，其他任何事件包括点击事件无法执行，需要等到滑动结束才能触发，这是IOSapp系统自带的，暂时无法解决

3、 问题：http和https的域名和接口api域名
如果H5的域名地址是http的，那接口api域名是没有限制的
如果H5的域名地址https的，那接口api域名必须要求是https的
如果H5是放在客户端app中使用的，那要看app内处理，看app有没有放开H5请求的权限，安卓客户端可以直接放开，但是IOS客户端需要配置证书之类的

css3 动画效果实现方法大大全
